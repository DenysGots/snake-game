# Snake Game
Simple snake game

Deployed on Heroku:
http://

Start local server:
* clone
* run `npm i`
* run `npm run dev:local`
* open browser at `http://localhost:5400`

### General:
* Angular 9, NestJS 7, Typescript, RxJS 6, Sass, Angular Material, Moment.js, Lodash
